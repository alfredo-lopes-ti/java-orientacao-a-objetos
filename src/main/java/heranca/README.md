# Herança em Java

## Conceitos praticados
- `extends` — herdar uma classe pai
- `super()` — chamar construtor do pai
- `@Override` — sobrescrever métodos
- Polimorfismo — `veiculos.heranca.Veiculo v = new veiculos.heranca.Carro(...)`

## Exemplos

### 🐾 animais/
Superclasse `Animal` com subclasses `Cachorro` e `Gato`.

### 🚗 veiculos/
Superclasse `veiculos.heranca.Veiculo` com subclasses `veiculos.heranca.Carro` e `veiculos.heranca.Moto`.
