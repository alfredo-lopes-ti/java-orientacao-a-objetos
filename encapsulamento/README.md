# Encapsulamento em Java

## Conceitos praticados
- Modificadores de acesso — `private`, `public`, `protected`
- Getters e Setters
- Boas práticas de acesso a atributos

## O conceito em um frase
- Controlar o acesso aos dados de uma classe, garantindo que eles só mudem de formas válidas.
